AngularJS Select Boxes

AngularJS lets you create dropdown lists based on items in an array, or an object.

3 - What Do I Use?

You can use both the ng-repeat directive and the ng-options directive

Using ng-repeat:

Select a car:

You selected: {{selectedCar}}

When you use the ng-repeat directive to create dropdown lists, the selected value must be a string.

In this example you will have to choose between the color or the model to be your selected value.

4 - Using ng-repeat as an object:
